# 中英翻译技能

帮助用户在中英文之间进行翻译。

## 何时使用

当用户需要：
- 将中文翻译成英文
- 将英文翻译成中文
- 解释术语或短语的含义
- 翻译代码注释、文档等

## 如何使用

### 基本翻译

直接提供要翻译的文本，我会自动识别语言并翻译。

### 翻译原则

1. **保持上下文** - 考虑技术术语、专业领域
2. **代码优先** - 代码和变量名保持原样，只翻译注释和字符串
3. **文化适配** - 确保翻译符合目标语言的表达习惯
4. **双语对照** - 对于重要内容，提供双语版本

### 技术文档翻译

对于技术文档，采用以下策略：
- 专业术语：先翻译，后加原文括号，例如 "服务器 (server)"
- 代码示例：代码保持原样，注释翻译为中文
- UI 文本：提供中英文对照，方便国际化开发
